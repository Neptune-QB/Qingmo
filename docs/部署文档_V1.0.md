## 三、服务端部署（3 步）

### 步骤 1：安装 Python 依赖

```bash
cd backend
pip install -r requirements.txt
```

核心依赖：`fastapi` `uvicorn` `pydantic` `pydantic-settings`

### 步骤 2：配置环境变量（可选）

```bash
# 如需 LLM 对话功能，编辑 .env 填入火山引擎 API Key
# 不配置也不影响基础高光互动功能
cp .env.example .env
# 编辑 .env 填入真实的 DOUBAO_API_KEY 和 DOUBAO_EP_ID
```

### 步骤 3：启动服务

```bash
# 数据库已预置数据，无需额外初始化
python main.py
```

服务启动后：
- API 地址：`http://127.0.0.1:8000`
- API 文档：`http://127.0.0.1:8000/docs`
- 健康检查：`http://127.0.0.1:8000/api/v1/health`

## 四、客户端部署

### Android 设备安装

1. 将 `app-release.apk` 传输到 Android 设备
2. 在设备上打开 APK 文件进行安装
3. 如提示"未知来源"，在设置中允许安装

### 模拟器安装

```bash
# 使用 adb 安装
adb install app-release.apk
```

### 网络配置

- 模拟器默认 `10.0.2.2` 映射到宿主机 `127.0.0.1`
- 如需在真机上运行，修改 `RetrofitClient.kt` 中的 `BASE_URL` 为电脑局域网 IP

## 五、演示流程

### 5.1 基础体验流程

1. 启动后端 `python main.py`
2. 打开 Android App
3. 首页展示 10 部短剧网格列表
4. 点击任一短剧进入详情页
5. 点击"播放"进入播放器

### 5.2 高光互动演示

以下 5 部短剧含有演示高光点数据（各有 12 条）：

| 短剧 | 高光点分布 |
|------|-----------|
| 天下第一纨绔 (ID=2) | 古装 — 7 集中有高光点 |
| 荒年全村啃树皮 (ID=5) | 年代 — 7 集中有高光点 |
| 云渺1 (ID=7) | 仙侠 — 7 集中有高光点 |
| 撕夜 (ID=8) | 悬疑 — 7 集中有高光点 |
| 那年冬至 (ID=9) | 校园 — 7 集中有高光点 |

**操作**：播放上述短剧，到达高光点时间点时：
- 小墨自动从右侧展开
- 情绪弹幕模式：展示 emoji 按钮组，点击后播放粒子特效
- 剧情投票模式：展示投票选项，选择后显示百分比动画
- 互动数据自动上报后端

### 5.3 对话功能演示

1. 在播放器中点击右侧小墨图标
2. 输入消息与 AI 对话
3. 支持流式逐字返回（需配置 LLM API Key）

### 5.4 管理功能演示

```bash
# 查看高光点数据版本
curl http://127.0.0.1:8000/api/v1/admin/highlights/version

# 批量上传新高光点
curl -X POST http://127.0.0.1:8000/api/v1/admin/highlights/batch \
  -H "Content-Type: application/json" \
  -d '[{"episode_id":1001,"time":45.0,"type":"sweet","title":"甜蜜时刻","widget_type":"emotion","emotion_hints":["甜哭了！","磕到了！"],"duration":15}]'
```

## 八、故障排查

| 问题 | 解决方案 |
|------|---------|
| 客户端无法连接后端 | 检查后端是否启动；模拟器 IP 是否为 10.0.2.2 |
| LLM 对话不可用 | .env 未配置 DOUBAO_API_KEY，不影响高光互动 |
| 高光点无数据 | 运行 `python seed_demo_highlights.py` |
| 播放器黑屏 | videos/ 目录下无实际视频文件，替换为真实 MP4 |
| 互动数据不显示 | 后端未启动或网络不通 |
