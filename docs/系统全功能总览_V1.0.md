# 青墨短剧平台 — 系统全功能总览 V1.0
> 生成日期：2026年6月7日 | 当前版本：1.0.0 生产就绪

---

## 一、系统概述

青墨是一个面向移动端的互动短剧全栈平台，核心特色是**真实物理232集MP4视频 + 全局AI小墨Agent + 高光点互动全链路闭环**，后端基于FastAPI+SQLite，Android端基于Jetpack Compose单Activity架构，全链路零Mock真实数据，已通过全量回归测试18/18用例通过，满足生产上线条件。

---

## 二、技术栈总览

| 层次 | 技术选型 | 版本 |
|------|---------|------|
| **后端框架** | FastAPI | 最新稳定版 |
| **数据库** | SQLite 原生 | 3.x |
| **LLM引擎** | 豆包 Doubao-Seed-2.0-lite (火山引擎Ark) | OpenAI兼容模式接入 |
| **客户端UI** | Jetpack Compose | Material3 |
| **播放器** | Media3 ExoPlayer | 1.2.1 |
| **网络层** | Retrofit2 + OkHttp | 2.9.0 / 4.12.0 |
| **图片加载** | Coil | 2.5.0 |
| **最低SDK** | Android minSdk 26 (Android 8.0) | targetSdk 34 |
| **代码规范** | ktlint | 1.3.1 |
