## 三、风险预案
| 风险点 | 影响等级 | 应对方案 |
|---|---|---|
| 大模型接口调用超时 | 中 | 内置本地兜底回复模板，离线模式下不影响基础高光互动功能使用 |
| 动画性能不达标 | 低 | V1.0优先使用Canvas轻量粒子特效，后续迭代替换Lottie精细动画 |
| 排期进度滞后 | 低 | 预留1天冗余缓冲时间，P2级功能（AI问答模块）可延后到比赛后V1.1版本迭代 |
